# NewIris Frontend

React + TypeScript + Vite SPA for WatchIO parameter monitoring.

## Quick start

```powershell
npm install
npm run dev
```

Open **http://localhost:5173** (not :8082 — that is the API gateway).

## Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Development server with HMR |
| `npm run build` | Production build to `dist/` |
| `npm run preview` | Preview production build |
| `npm run lint` | ESLint |

## Configuration

Copy `.env.example` to `.env` and adjust as needed. See the root [README.md](../README.md) and [docs/NewIrisConnection.md](../docs/NewIrisConnection.md) for connection details.
